## 📦 下载安装

**[👉 点击下载最新版](https://github.com/DrowTT/lf-todo/releases/latest)**

> 下载 `lf-todo-x.x.x-setup.exe` 运行安装即可。应用内置自动更新，后续新版本会自动提醒。

## ✨ 功能特性

- 📝 待办增删改查，支持子任务
- 📂 多分类管理，Ctrl+数字键快速切换
- 🔀 拖拽排序任务
- ⌨️ 自定义快捷键绑定
- 🔄 应用内自动更新
- 💾 窗口大小/位置自动记忆
- 🗑️ 已完成任务自动清理
- 📤 数据导出为 JSON
- 🖥️ 系统托盘常驻，开机自启

## 🎨 界面预览

Arctic Blue 主题 · 无边框窗口 · 玻璃态设计

## 开发

```bash
# 安装依赖
pnpm install

# 开发
pnpm dev

# 打包
pnpm build:win
```
